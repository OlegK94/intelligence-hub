# 🧪 Inhaltsstoffe — Datenbank

> Zielgruppe: Sportler, Gründer, High Achiever — wenig Zeit, maximaler Effekt, verlässlich, kein Prozedere.
> Bewertung nach: Wirksamkeit, Sicherheit, Kombinierbarkeit, regulatorische Lage (EU/DACH).

---

## Bewertungsmatrix

| Symbol | Bedeutung |
|---|---|
| ✅ | Stark belegt / empfohlen |
| 🟡 | Moderat belegt / situativ |
| ⚠️ | Risiko / Vorsicht |
| ❌ | Nicht empfohlen für dieses Produkt |

---

## 1. Koffein

| Feld | Detail |
|---|---|
| **Kategorie** | Stimulans (Xanthin) |
| **Wofür bekannt** | Wachheit, Fokus, Ausdauer, Fettstoffwechsel |
| **Dosis (effektiv)** | 100–200 mg pro Einheit (Performance) |
| **Max/Tag** | 400 mg (EFSA/FDA) |
| **Wirkdauer** | 4–6 Stunden (HWZ ~5h) |
| **Vorteile** | Schnell wirkend, gut erforscht, legal, skalierbar |
| **Nachteile** | Toleranzentwicklung, Entzug (Kopfschmerz), schlechter Schlaf bei Spätkonsum, Anxiety bei Überdosis |
| **Risiken** | Herzrhythmusstörungen bei >600 mg/Tag, Wechselwirkung mit bestimmten Medikamenten |
| **Bewertung** | ✅ Basis jedes Performance-Kaffees |

---

## 2. L-Theanin

| Feld | Detail |
|---|---|
| **Kategorie** | Aminosäure (natürlich in Tee) |
| **Wofür bekannt** | "Smooth Focus" — Koffein ohne Jitter, Alpha-Wellen im Gehirn |
| **Dosis (effektiv)** | 100–200 mg (1:1 bis 2:1 zu Koffein) |
| **Max/Tag** | 1.200 mg gut toleriert |
| **Wirkdauer** | 3–5 Stunden |
| **Vorteile** | Eliminiert Koffein-Jitter, verbessert Aufmerksamkeit + Reaktionszeit, kein Crash, synergistisch |
| **Nachteile** | Allein kaum spürbar, muss mit Koffein kombiniert werden |
| **Risiken** | Praktisch keine — eines der sichersten Supplemente überhaupt |
| **Goldene Kombination** | **Koffein 100 mg + L-Theanin 200 mg** = stärkster belegter kognitiver Stack |
| **Bewertung** | ✅ Pflicht-Zutat #1 — höchstes Nutzen/Risiko-Verhältnis aller Optionen |

---

## 3. L-Tyrosin

| Feld | Detail |
|---|---|
| **Kategorie** | Aminosäure (Dopamin-Vorläufer) |
| **Wofür bekannt** | Fokus unter Stress, kognitive Leistung bei Schlafmangel, Entscheidungsgeschwindigkeit |
| **Dosis (effektiv)** | 500–2.000 mg |
| **Max/Tag** | 12 g (klinisch) — praktisch 2–4 g |
| **Wirkdauer** | 2–3 Stunden |
| **Vorteile** | Besonders wirksam bei Stress, Zeitdruck, Schlafentzug — ideal für High Achiever |
| **Nachteile** | Wenig Effekt bei ausgeruhten Personen, Wirkung variiert stark individuell |
| **Risiken** | Bei Schilddrüsenerkrankungen kontraindiziert, MAO-Hemmer-Wechselwirkung |
| **Bewertung** | ✅ Top-Kandidat für unsere Zielgruppe (Stress + Time Pressure = genau das) |

---

## 4. Alpha-GPC

| Feld | Detail |
|---|---|
| **Kategorie** | Cholin-Verbindung (Acetylcholin-Vorläufer) |
| **Wofür bekannt** | Kognitive Leistung, Reaktionszeit, Power Output beim Training |
| **Dosis (effektiv)** | 300–600 mg |
| **Wirkdauer** | 4–6 Stunden |
| **Vorteile** | Einziges Supplement das gleichzeitig kognitive UND physische Performance steigert, gut belegt |
| **Nachteile** | Teuer, unangenehmer Eigengeruch in Pulverform, fischartig wenn nicht verkapselt |
| **Risiken** | Mögliche Assoziation mit erhöhtem Schlaganfallrisiko bei sehr hohen Langzeitdosen (Studienlage gemischt) |
| **Regulatorisch** | Als Novel Food in EU eingeschränkt — max 400 mg/Tag in Lebensmitteln |
| **Bewertung** | 🟡 Sehr wirksam, aber Geschmack + EU-Regulatorik prüfen |

---

## 5. Ashwagandha (KSM-66 oder Sensoril)

| Feld | Detail |
|---|---|
| **Kategorie** | Adaptogen (ayurvedisch) |
| **Wofür bekannt** | Stressreduktion, Cortisol-Senkung, Schlafqualität, Ausdauer |
| **Dosis (effektiv)** | 300–600 mg (KSM-66 Extrakt) |
| **Wirkdauer** | Akkumulativ — Wirkung nach 4–8 Wochen Einnahme |
| **Vorteile** | Reduziert gefühlten Stress signifikant, verbessert Erholung, Testosteron moderat |
| **Nachteile** | Keine Akutwirkung (falsche Erwartung!), Geschmack erdig/bitter |
| **Risiken** | Lebertoxizität bei sehr hohen Dosen (Fallberichte), Wechselwirkung mit Schilddrüsenmedikamenten |
| **Bewertung** | 🟡 Guter Basis-Stack für langfristige Nutzer — nicht als "sofort"-Inhaltsstoff positionieren |

---

## 6. Rhodiola Rosea

| Feld | Detail |
|---|---|
| **Kategorie** | Adaptogen |
| **Wofür bekannt** | Anti-Fatigue, mentale Ausdauer, Burnout-Prävention |
| **Dosis (effektiv)** | 200–600 mg (mind. 3% Rosavine + 1% Salidroside) |
| **Wirkdauer** | Akut spürbar (schneller als Ashwagandha), akkumulativ |
| **Vorteile** | Eine der wenigen Adaptogene mit akutem Effekt — passt zur "sofort"-Positionierung |
| **Nachteile** | Qualität stark schwankend je nach Extrakt, Geschmack bitter |
| **Risiken** | Stimulierend — bei Abend-Einnahme Schlafstörungen möglich |
| **Bewertung** | ✅ Bester Adaptogen-Kandidat für unsere Zielgruppe (akute Wirkung!) |

---

## 7. Lion's Mane (Hericium erinaceus)

| Feld | Detail |
|---|---|
| **Kategorie** | Funktionspilz |
| **Wofür bekannt** | BDNF-Förderung, Neuroplastizität, "Brain fog" reduzieren |
| **Dosis (effektiv)** | 500–1.000 mg (Fruchtköper-Extrakt, mind. 25% Beta-Glucane) |
| **Wirkdauer** | Akkumulativ (Wochen) |
| **Vorteile** | Trendy, gut vermarktbar, zunehmend wissenschaftlich belegt |
| **Nachteile** | Wenig Akutwirkung, Qualität stark schwankend, teuer |
| **Risiken** | Selten allergische Reaktionen, sonst gut verträglich |
| **Bewertung** | 🟡 Marketingpotenzial hoch, aber "Performance sofort" passt nicht zum Wirkmechanismus |

---

## 8. B-Vitamine (B6, B12, Pantothensäure)

| Feld | Detail |
|---|---|
| **Kategorie** | Vitamine |
| **Wofür bekannt** | Energiestoffwechsel, Nervenfunktion, Reduktion von Müdigkeit |
| **Dosis (effektiv)** | EFSA NRV: B6 1,4 mg / B12 2,5 μg / B5 6 mg |
| **Vorteile** | Wasserlöslich (kein Überdosierungsrisiko), günstig, EU Health Claims vorhanden |
| **Nachteile** | Wenig spürbare Akutwirkung bei ausreichend Versorgten |
| **Risiken** | B6 Neuropathie bei extremer Überdosierung (>200 mg/Tag, kein Produkt-Risiko) |
| **Regulatorisch** | ✅ EU Health Claims erlaubt ("trägt zur Verringerung von Müdigkeit bei") |
| **Bewertung** | ✅ Pflicht-Ergänzung — sicher, legal, vermarktbar |

---

## 9. Elektrolyte (Na, Mg, K)

| Feld | Detail |
|---|---|
| **Kategorie** | Mineralstoffe |
| **Wofür bekannt** | Hydration, Muskelkontraktion, Kognition bei Dehydration |
| **Vorteile** | Koffein ist diuretisch → Elektrolyte kompensieren, direkt spürbar |
| **Bewertung** | ✅ Sinnvoll für Sport-Positionierung, unterscheidet von Standard-Kaffee |

## Kombinationsmatrix — Empfohlene Stacks

### Stack A — "Focus Shot" (Basis-Produkt)
```
Koffein       100 mg   ← aus Kaffee
L-Theanin     200 mg
L-Tyrosin     500 mg
B-Vitamine    NRV
```
**Profil:** Klarer Kopf, kein Jitter, Stress-resistent. Für tägliche Nutzung.

---

### Stack B — "Deep Work" (Premium-Produkt)
```
Koffein       150 mg   ← aus Kaffee
L-Theanin     300 mg
L-Tyrosin     1.000 mg
Rhodiola      300 mg
B-Vitamine    NRV
```
**Profil:** Maximale kognitive Leistung unter Druck. Für intensive Arbeitsphasen.

---

### Stack C — "Pre-Training" (Sport-Produkt)
```
Koffein       200 mg   ← aus Kaffee
L-Theanin     200 mg
Alpha-GPC     300 mg   (EU-Grenze beachten)
Elektrolyte   Standard
B-Vitamine    NRV
```
**Profil:** Physische + mentale Performance vor dem Training.
