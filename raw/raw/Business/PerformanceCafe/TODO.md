## 🔴 PHASE 0 — Sofort (Jetzt, Research-Phase)

### Regulatorik klären — BLOCKER für Stack-Entscheidung
- [ ] **NMN EU Novel Food Status 2025/2026 verbindlich klären**
  → BfR.de oder EFSA-Datenbank direkt prüfen
  → Alternativ: Regulatory-Affairs-Berater beauftragen (~500–1.500 €)
  → Wenn nicht zugelassen: NMN aus Tier-1 streichen, Trigonellin (natürlich im Kaffee) als NAD+-Argument nutzen

- [ ] **Urolithin A — Amazentis/Timeline Patent prüfen**
  → Läuft das Patent in EU? Lizenz möglich?
  → Alternative: Urolithin A selbst als Novel Food anmelden (lang + teuer) ODER weglassen
  → Entscheidung: Tier-2 SKU erst wenn geklärt

- [ ] **Spermidine aus Weizenkeimen — Novel Food Status DE**
  → In DE als Lebensmittel oder NEM?

- [ ] **Lebensmittel vs. NEM entscheiden**
  → NEM-Weg: schneller, flexibler, aber Einschränkungen bei Health Claims
  → Empfehlung aus Modul 5: NEM-Weg für MVP

### Name & Marke finalisieren
- [ ] **Top-3 Namen aus naming_brief.md prüfen:** AEVUM / KOLVR / ORVYN
  → .com Domain-Verfügbarkeit prüfen (namecheap.com oder instant.domains)
  → EUIPO Datenbank: eSearch plus (euipo.europa.eu) — Klasse 30 + Klasse 5
  → Entscheidung treffen: welcher Name?

- [ ] **EUIPO Markenanmeldung starten**
  → Kosten: ~1.000 € für 2 Klassen
  → Anmeldung VOR erstem öffentlichem Auftreten
  → Anwalt oder direkt über euipo.europa.eu

### Prototyp-Entwicklung starten
- [ ] **CMO kontaktieren** (aus research/05_prototyp_partner.md)
  → Anfrage: Machbarkeit Instant-Sachet mit funktionellen Inhaltsstoffen
  → MOQ, Kosten, Timeline erfragen
  → Fokus: NAC-Geruchsmaskierung + Curcumin-Löslichkeit klären

- [ ] **Kaffee-Basis definieren**
  → Light-Medium Roast Arabica/Robusta Blend festlegen
  → Vietnamesischer Robusta (Dak Lak) als Differenzierungsmerkmal?
  → Sample-Bestellung bei EU-Importeur

- [ ] **Tier-1 Stack final definieren** (ohne regulatorische Risiken)
  → Morning Performance: Kaffee + L-Theanin + Alpha-GPC + Lion's Mane + Ashwagandha + Kreatin + Taurin
  → Dosierungen aus research/02_ingredienzen_db.md bestätigen

  → Budget: ~200 € (Domain + Tool-Kosten Monat 1)

- [ ] **Waitlist-Ziel:** 500 Signups in 60 Tagen = Go-Signal für Produktion

  → Standgebühr anfragen (~3.000 €)
  → Sampling-Konzept: Sachet-Probe + QR-Code zur Waitlist
