# 📐 ALG I – Progressionsvorbehalt

## Funktionsweise (§32b EStG)

1. ALG I (Lohnersatzleistung) ist selbst **steuerfrei**
2. Das ALG I wird jedoch dem übrigen Einkommen **fiktiv hinzugerechnet**
3. Daraus ergibt sich ein höherer **Grenzsteuersatz**
4. Dieser erhöhte Satz wird dann auf das **tatsächlich steuerpflichtige Einkommen** angewendet

## Beispiel 2025 (geschätzt)

| | |
|---|---|
| Lohnsteuerpfl. Einkommen | ~30.000 € |
| ALG I (fiktiv hinzugerechnet) | ~8.400 € |
| Fiktives Gesamteinkommen | ~38.400 € |
| Steuersatz auf 38.400 € | ~25% |
| **Steuer auf echte 30.000 €** | **~7.500 €** |
| vs. ohne Progressionsvorbehalt | ~5.500 € |
| **Mehrbelastung ca.** | **~2.000 €** |

> Genaue Berechnung durch SP STB.
