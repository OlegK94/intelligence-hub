## 3. Kritische Risikohinweise

### 🔴 Insolvenzrisiko (Beide Unternehmen)
- **§17 InsO** (Zahlungsunfähigkeit): Unfähigkeit, fällige Verbindlichkeiten zu begleichen
- **§19 InsO** (Überschuldung): Verbindlichkeiten übersteigen Vermögen (negatives Eigenkapital)
- **Status:** Beide Unternehmen erfordern eine sofortige Dokumentenprüfung (Jahresabschlüsse, BWA, Kontoauszüge, Gesellschaftsvertrag)
- Bis zur Prüfung der Unterlagen kann eine Insolvenz für keine der beiden Einheiten ausgeschlossen werden

### 🔴 GF-Haftung (Doppelexposition)
- Oleg ist Geschäftsführer (GF) **beider** Unternehmen — Wagglz GmbH und OK Capital UG
- **§17/19 InsO:** Sobald eine Insolvenz festgestellt ist, hat der GF **21 Tage** zur Antragstellung (Antragspflicht)
- Unterlassung → strafrechtliche Haftung wegen **Insolvenzverschleppung**
- Das persönliche Haftungsrisiko ist existenziell, falls eine Insolvenz technisch bereits vorliegt

### 🔴 Gesellschafterdarlehen-Risiko
- **§39 InsO:** Gesellschafterdarlehen sind im Insolvenzfall nachrangig (werden zuletzt bedient)
- **§135 InsO:** 10-jähriges Anfechtungsrisiko — Rückzahlungen von Gesellschafterdarlehen innerhalb von 10 Jahren vor der Insolvenz können rückgängig gemacht werden
- Die laufenden ~357 €/Monat Gesellschafterdarlehen-Zahlungen erzeugen eine fortlaufende Exposition

### 🟡 Steuerstatus
- Einreichungsstatus der Steuererklärungen unbestätigt
- Mögliche Verlustvorträge ungenutzt — Wert nicht realisiert
- Ungeklärtes Steuerbild verstärkt das Gesamtrisiko

## 4. Rehabilitationsplan (Drei Phasen)

(Quelle: [[Rehabilitation Plan]], [[Finanz-Übersicht Source Detail]])

### Phase 1: Sofortstabilisierung (Priorität: JETZT)
1. Vollständige Dokumentenprüfung für beide Unternehmen (BWA, Jahresabschlüsse, Gesellschaftsvertrag)
2. Insolvenzprüfung für jede Einheit (§17/§19 InsO-Test)
3. GF-Haftungsexposition einschätzen
4. Rangrücktritt-Status bestehender Gesellschafterdarlehen klären

### Phase 2: Juli-Provisions-Zuweisung
Prioritätsreihenfolge für die ~4.611 € Juli-Provision:
1. Rückzahlung Steuerberaterhonorar
2. Dispo-Ausgleich (−1.405 € Überziehung ausgleichen)
3. Notfall-Puffer (Ziel: 1 Monatsgehalt)
4. Rest: strukturiertes Defizitmanagement

### Phase 3: Strukturelle Lösung
- Strategische Optionen für jedes Unternehmen:
  - **Wagglz GmbH:** Fortführen (wenn solvent) / restrukturieren / abwickeln
  - **OK Capital UG:** Liquidieren / Ruhend halten / in Wagglz eingliedern
- Wiederkehrenden Sparpuffer aufbauen
- Strukturelles Defizit durch Einkommenserhöhung (DoktorLib-Beförderung, Nebeneinkünfte) oder Kostenreduzierung abbauen
