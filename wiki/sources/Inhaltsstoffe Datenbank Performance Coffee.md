# Inhaltsstoffe Datenbank — Performance Coffee Brand

## Überblick

Dieses Quelldokument ist die zentrale Inhaltsstoff-Datenbank für die [[Performance Coffee Brand]] (intern unter [[Wagglz GmbH]] / Performance Coffee Brand). Es enthält eine strukturierte Bewertungsmatrix für 9 Kerninhaltsstoffe sowie drei empfohlene Produktstacks.

**Zielgruppe des Produkts:** Sportler, Gründer, High Achiever — wenig Zeit, maximaler Effekt, verlässlich, kein Prozedere.

**Bewertungskriterien:** Wirksamkeit, Sicherheit, Kombinierbarkeit, regulatorische Lage (EU/DACH).

> Für Einzel-Konzeptseiten der Inhaltsstoffe siehe: [[Koffein Performance Coffee]], [[L-Theanin Performance Coffee]], [[L-Tyrosin Performance Coffee]], [[Alpha-GPC]], [[Ashwagandha KSM-66]], [[Rhodiola Rosea Performance Coffee]], [[Lion's Mane Performance Coffee]], [[B-Vitamine Performance Coffee]], [[Supplement Stacks Performance Coffee]]

## 1. Koffein

| Feld | Detail |
|---|---|
| **Kategorie** | Stimulans (Xanthin) |
| **Dosis (effektiv)** | 100–200 mg/Einheit |
| **Max/Tag** | 400 mg (EFSA/FDA) |
| **Wirkdauer** | 4–6 Stunden (HWZ ~5h) |
| **Bewertung** | ✅ Basis jedes Performance-Kaffees |

**Risiken:** Toleranzentwicklung, Entzug, schlechter Schlaf bei Spätkonsum; Herzrhythmusstörungen >600 mg/Tag.

**EU Health Claims:** ✅ „trägt zur Steigerung der Konzentration bei"

## Empfohlene Stacks

### Stack A — "Focus Shot" (Basis-Produkt)
```
Koffein       100 mg   (aus Kaffee)
L-Theanin     200 mg
L-Tyrosin     500 mg
B-Vitamine    NRV
```
**Profil:** Klarer Kopf, kein Jitter, Stress-resistent. Für tägliche Nutzung.

---

### Stack B — "Deep Work" (Premium-Produkt)
```
Koffein       150 mg   (aus Kaffee)
L-Theanin     300 mg
L-Tyrosin     1.000 mg
Rhodiola      300 mg
B-Vitamine    NRV
```
**Profil:** Maximale kognitive Leistung unter Druck. Für intensive Arbeitsphasen.

---

### Stack C — "Pre-Training" (Sport-Produkt)
```
Koffein       200 mg   (aus Kaffee)
L-Theanin     200 mg
Alpha-GPC     300 mg   (EU-Grenze beachten)
Elektrolyte   Standard
B-Vitamine    NRV
```
**Profil:** Physische + mentale Performance vor dem Training.
