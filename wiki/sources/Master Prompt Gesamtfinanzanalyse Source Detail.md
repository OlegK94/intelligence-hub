Master Prompt Gesamtfinanzanalyse ist der Prompt-Template für neue Claude-Gesprächs-Sessions zur integrierten Finanzanalyse von [[Oleg Personal Context]]. Der Prompt ist direkt in ein neues Claude-Gespräch zu kopieren, dann werden die Dokumente hochgeladen (source: [[Master Prompt Gesamtfinanzanalyse Source Detail]]).

## Verwendung

1. Gesamten Prompt-Block kopieren
2. Neues Claude-Gespräch starten
3. Prompt einfügen
4. Dokumente hochladen (Checkliste im Dokument)

Alternativ: [[Claude Session Template Performance]] für Performance-Themen.

## Vier Expertenperspektiven

| Kürzel | Rolle | Fokus |
|---|---|---|
| [FD] | Finance Director | Zahlen, Strukturen, Cashflow, konsolidierte Sicht |
| [ST] | Steuerberater | Steuerliche Risiken, Optimierung, Compliance |
| [IA] | Insolvenzanalyst | Überschuldung, Zahlungsunfähigkeit, Haftung |
| [US] | Unternehmensstratege | Entscheidungen, Prioritäten, Handlungsoptionen |

## Fünf Analyse-Schritte

1. **Insolvenzprüfung [IA]** — §17 InsO (Zahlungsunfähigkeit) + §19 InsO (Überschuldung) für jede Gesellschaft; Antragspflicht; Haftungsrisiken
2. **Steuerliche Lage [ST]** — Offene Schulden, fehlende Erklärungen, Verlustvorträge, Gesellschafterdarlehen
3. **Konsolidierte Vermögenslage [FD]** — Alle Aktiva/Passiva, persönliche Haftungsexponierung
4. **Unternehmens-Rentabilität [US]** — Je Gesellschaft: Umsatz, Break-Even, Exit-Optionen
5. **Gesamtstrategie** — Reihenfolge, Zeitplan, Weg zur finanziellen Stabilität

## Sechs Deliverables

1. Ampel-Dashboard (ROT/GELB/GRÜN je Entität mit Begründung)
2. Konsolidierte Bilanz (alle Aktiva/Passiva)
3. Steuerliche To-Do-Liste mit Fristen
4. Sofortmaßnahmen 30-Tage-Plan
5. Strategische Optionen je Gesellschaft (3 Szenarien)
6. Gesamtstrategie + Zeitplan

## Dokumente-Checkliste

**Privat:** Kontoauszüge CSV (Consorsbank), ESt-Erklärungen 2021–2024, VW-Bank Kreditvertrag
**Wagglz:** BWA, Jahresabschlüsse, Kontoauszüge, Steuerbescheide
**OK Capital:** Jahresabschluss/BWA, Kontoauszüge
**Verträge:** Gesellschaftsverträge, Darlehensverträge, Rangrücktritte

## Kontext-Block (vorkonfiguriert)

Der Prompt enthält einen vollständig ausgefüllten Kontext-Block mit allen bekannten Finanzdaten von Oleg — damit beginnt jede neue Analyse-Session ohne Wiederholung bereits bekannter Fakten.
