## Priorität 1 — Kernmarker (aus Quelle)

| Marker | Optimaler Zielbereich | Warum |
|---|---|---|
| ApoB | <70 mg/dL | Kardiovaskuläres Risiko — wichtiger als LDL-C |
| Nüchtern-Glucose | 70–85 mg/dL | Metabolische Gesundheit |
| HbA1c | <5,2 % | Langzeit-Glucosekontrolle |
| Nüchtern-Insulin | <5 µIU/mL | Insulinsensitivität |
| hsCRP | <0,5 mg/L | Systemische Inflammation |
| Testosteron (gesamt) | 600–900 ng/dL | Hormonstatus, Muskelaufbau |
| SHBG | 20–40 nmol/L | Freies Testosteron berechnen |
| Freies Testosteron | >15 ng/dL | Aktiver Anteil |
| IGF-1 | 150–250 ng/mL | Wachstumshormonachse |
| TSH | 1,0–2,0 mIU/L | Schilddrüsenfunktion |
| Vitamin D (25-OH) | 50–80 ng/mL | Immunfunktion, Performance |

### Wichtiger Hinweis aus der Quelle (wörtlich)

> [!important] ApoB separat anfordern
> ApoB ist **nicht** in Standard-Blutpanels enthalten. Explizit beim Arzt anfordern. Wichtiger kardiovaskulärer Marker als LDL-C — misst die tatsächliche Partikelzahl.
