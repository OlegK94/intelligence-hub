# Hyrox — 10-Week Training Plan

## Überblick

Dies ist ein **strukturiertes 10-Wochen-Trainings-Programm** zur Vorbereitung auf den [[Hyrox]]-Wettkampf. Das Programm nutzt progressive Phasierung (Wochen 1–3, 4–6, 7–9, 10) und integriert [[Zone 2 Training]], Kraftentwicklung und [[Brick Training]] zum Trainieren des "Gear Switch" zwischen Stationen und Lauf.

**Ziel-Finish-Zeit:** 1:35–1:45h (realistisch für Anfänger mit guter Kraft-Base)
**Primärer Limitor:** Aerobe Kapazität (Zone 2)

---

## Athleten-Profil (Input)

- **Stärke:** Sehr gut — 125 kg Sled Push/Pull nach 100 Wall Balls
- **Schwäche:** Aerobe Kapazität, Laufausdauer
- **5 km Lauf-Zeit (geschätzt):** ~30 min (6:00 min/km)
- **Realistische Finish-Zeit:** 1:35–1:45h (unter Annahme von 12-Wochen Zone-2-Basis-Aufbau)

---

## Weekly Training Structure (5 Tage)

| Tag | Einheit | Dauer | Fokus |
|---|---|---|---|
| **Montag** | Kraft + Sled | 60 min | Sled Push/Pull, Kraft-Ausdauer |
| **Dienstag** | Easy Run mit Hund | 35 min | Zone 1–2 Recovery Run |
| **Mittwoch** | Rest | — | Vollständige Erholung |
| **Donnerstag** | Hyrox Simulation Circuit | 60 min | Stationen + Lauf, Brick Training |
| **Freitag** | Rest | — | Vollständige Erholung |
| **Samstag** | Kraft + Conditioning | 55 min | Kraft-Ausdauer, Metabolic Conditioning |
| **Sonntag** | Zone-2 Run mit Hund | 40 min | Zone 2 Steady State, aerobe Basis |

**Gesamtvolumen:** ~5 Trainings-Tage/Woche, 250 min/Woche strukturiertes Training

---

## 10-Wochen-Phasierung

### Phase 1 — Wochen 1–3: Aerobe Basis
**Ziel:** Laufpensum aufbauen, Zone 2 Kapazität etablieren

- Fokus: Zone 2 Lauf, Stationen mit **leichtem Gewicht** + Technik-Fokus
- Kein Wettkampftempo
- Kein AMRAP (As Many Reps As Possible) — kontrollierte Reps
- **Run-Struktur:** 35 min Easy + 40 min Zone-2 Steady (75 min/Woche Lauf)
- **Kraft:** Basis-Übungen (Squats, Rows, Sled-Bewegung mit ~50% Wettkampfgewicht)

**Trainings-Beispiel Woche 1:**
- Mo: Squat 5×5 @ 70%, Sled Push 60 kg × 3 Sets × 50m
- Di: 35 min easy run Zone 1
- Do: Ski Erg 500m, Sled Push 60 kg × 3, Sled Pull 50 kg × 3, Farmers Carry 200m, Run 200m — 3 Rounds
- Sa: Deadlift 3×3 @ 75%, Row 5×5, Wall Balls 6kg 50 Reps
- So: 40 min Zone 2 steady state run

### Phase 2 — Wochen 4–6: Spezifisches Conditioning
**Ziel:** Brick Training einführen, Intensität steigern, Wettkampfgewichte implementieren

- Fokus: **Brick Training** einführen (Station → 200m Run → nächste Station)
- Sled-Gewichte auf **102 kg (Push) & 78 kg (Pull)** erhöhen
- Lauftempo progressiv steigern (easy → Zone 2 → Zone 2 High)
- **Run-Struktur:** 35 min easy (Di) + 40 min Zone 2 (So), mit Brick-Übergängen (Do)

**Trainings-Beispiel Woche 5:**
- Mo: Sled Push 102 kg × 5 Sets × 50m, Squat 5×5 @ 80%, Rows 5×5
- Di: 35 min easy run
- Do: Ski Erg 750m, Sled Push 102 kg × 50m → Run 200m, Sled Pull 78 kg × 50m → Run 200m, Burpee Broad Jump 40m, Farmers Carry 100m → Run 200m (3 Rounds)
- Sa: Deadlift 3×3 @ 80%, Row 5×5, Wall Balls 6kg 75 Reps
- So: 40 min Zone 2 steady state run

### Phase 3 — Wochen 7–9: Race Simulation
**Ziel:** Vollständige Hyrox-Simulation trainieren, Wettkampftempo etablieren

- Fokus: **Vollständige Hyrox-Simulation 1× pro Woche** (Donnerstag)
- Wettkampftempo üben (konstant, nicht zu schnell starten)
- **Laufstrecken auf 1 km verlängern** (vollständige Station→1km Run Pattern)
- Taper-Vorbereitung (etwas Volumen-Reduktion in W9)

**Trainings-Beispiel Woche 7 (Full Simulation Tag):**
- Mo: Sled Push 102 kg × 3 Sets, Squat 3×5 @ 85%, Short-Brick-Circuits
- Di: 30 min easy run
- Do: **FULL HYROX SIMULATION**
  - Ski Erg 1.000m → Run 1 km
  - Sled Push 102 kg → Run 1 km
  - Sled Pull 78 kg → Run 1 km
  - Burpee Broad Jump 80m → Run 1 km
  - Rowing 1.000m → Run 1 km
  - Farmers Carry 2×24kg 200m → Run 1 km
  - Sandbag Lunges 20kg 100m → Run 1 km
  - Wall Balls 6kg 100 Reps → Run 1 km
  - **Ziel-Zeit:** 1:35–1:50h (Anfänger), mit Pace-Anpassung
- Sa: Kraft-Ausdauer-Circuits (Reduktion volume, Intensität halten)
- So: 35 min Zone 2 run

### Phase 4 — Woche 10: Taper
**Ziel:** Regeneration priorisieren, frisch für Wettkampf ankommen

- **Volumen -40%** (reduzieren, aber nicht auf Null)
- **Intensität erhalten** (einige schnelle Wiederholungen, aber kürzer)
- Regeneration Fokus (Schlaf, Ernährung, Stress-Management)
- **Logistik klären:** Anmeldung final, Ausrüstung prüfen, Strecken-Besichtigung (falls möglich), Ablauf-Plan

**Trainings-Beispiel Woche 10:**
- Mo: Sled Push 102 kg × 2 Sets × 30m, Squat 3×3 @ 85% (kurz)
- Di: 20 min easy run
- Do: Mini Hyrox-Simulation (Stationen 1–4 nur, 50% Volume)
- Sa: Kraft-Ausdauer 2×5 (kurz, Intensität erhalten)
- So: 25 min Zone 2 run (einfach, erholsam)
- **Mittwoch + Freitag:** Vollständige Ruhe (oder Foam Rolling, Mobility nur)

## Brick Training — Konzept (Core Method)

**Definition:** Direkt nach einer Station in den Lauf gehen — kein Stop, kein Reset, kein Active Recovery.

**Physiologische Rationale:** Trainiert den "Gear Switch" von anaerob (Station) → aerob (Lauf), die im Wettkampf **kritisch** ist. Athleten ohne Brick-Training erleben typischerweise "Dead Leg" oder massive Leistungs-Drops bei den Übergängen.

**Anwendung im Plan:**
- Phase 1: Kein Brick (Basis-Aufbau)
- Phase 2: Kurz-Bricks (200m nach Stationen)
- Phase 3: Mittel- bis Lang-Bricks (400m–1km nach Stationen)
- Phase 4: Mini-Bricks (Taper, Frische erhalten)

Siehe [[Brick Training]] für vollständige Methoden-Details.

## Primäre Limitierungen & Fokus

### 🔴 Aerobe Kapazität (HAUPTLIMITOR)
Das Profil zeigt: Sehr gute Kraft (125 kg Sled Push/Pull), aber schwache Ausdauer (5 km ≈ 30 min = 6:00/km, Zone-2-Tempo).

**Lösung:** Zone-2-Fokus in Phasen 1–3 (75 min/Woche Lauf, progressiv zu 100+ min)
- Phase 1: 75 min/Woche Lauf (35 easy + 40 Zone 2)
- Phase 2: 80–90 min/Woche (easy + Zone 2 + Bricks)
- Phase 3: 100+ min/Woche (Volles Circuit-Volumen + 1 km Bricks)

### 🟡 Kraft-Ausdauer (SEKUNDÄR)
Kraft ist vorhanden, aber Wiederholungs-Kapazität (z.B. 100 Wall Balls, 200m Farmers Carry) erfordert spezifisches Training.

**Lösung:** Kraft-Ausdauer-Fokus (Reps mit Tempo, nicht 1RM-Kraft)
- Squat 5×5 @ 75–85% (statt 1RM-Test)
- Wall Balls 50–100 Reps (statt 10 Reps max)
- Farmers Carry mit Bewegung (nicht statisch halten)
