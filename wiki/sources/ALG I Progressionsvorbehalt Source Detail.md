## Overview

## Funktionsweise (§32b EStG)

The source describes the mechanism in four steps:

1. ALG I (Lohnersatzleistung) ist selbst **steuerfrei**
2. Das ALG I wird jedoch dem übrigen Einkommen **fiktiv hinzugerechnet**
3. Daraus ergibt sich ein höherer **Grenzsteuersatz**
4. Dieser erhöhte Satz wird dann auf das **tatsächlich steuerpflichtige Einkommen** angewendet

This description is fully consistent with the existing [[ALG I Progressionsvorbehalt]] concept page — no contradictions.

## Beispiel 2025 (from Source)

| | |
|---|---|
| Lohnsteuerpfl. Einkommen | ~30.000 € |
| ALG I (fiktiv hinzugerechnet) | ~8.400 € |
| Fiktives Gesamteinkommen | ~38.400 € |
| Steuersatz auf 38.400 € | ~25% |
| **Steuer auf echte 30.000 €** | **~7.500 €** |
| vs. ohne Progressionsvorbehalt | ~5.500 € |
| **Mehrbelastung ca.** | **~2.000 €** |

> ⚠️ **Note:** The example uses ~30.000 € as the lohnsteuerpflichtige income base. This is a **schematic estimate** — the actual Wagglz GF salary figure for 2025 differs (see [[Wagglz GF-Gehalt 2025]]). The example is illustrative of the mechanism, not a precise calculation. As the source itself notes: *"Genaue Berechnung durch SP STB."*
