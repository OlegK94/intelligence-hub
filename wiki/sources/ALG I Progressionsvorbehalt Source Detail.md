## Funktionsweise (§32b EStG)

Beide Quellen beschreiben den Mechanismus identisch in vier Schritten:

1. ALG I (Lohnersatzleistung) ist selbst **steuerfrei**
2. Das ALG I wird jedoch dem übrigen Einkommen **fiktiv hinzugerechnet**
3. Daraus ergibt sich ein höherer **Grenzsteuersatz**
4. Dieser erhöhte Satz wird dann auf das **tatsächlich steuerpflichtige Einkommen** angewendet

Diese Beschreibung stimmt vollständig mit der bestehenden Konzeptseite [[ALG I Progressionsvorbehalt]] überein — keine Widersprüche.

## Beispiel 2025 (aus beiden Quellen — identisch)

| | |
|---|---|
| Lohnsteuerpfl. Einkommen | ~30.000 € |
| ALG I (fiktiv hinzugerechnet) | ~8.400 € |
| Fiktives Gesamteinkommen | ~38.400 € |
| Steuersatz auf 38.400 € | ~25% |
| **Steuer auf echte 30.000 €** | **~7.500 €** |
| vs. ohne Progressionsvorbehalt | ~5.500 € |
| **Mehrbelastung ca.** | **~2.000 €** |

> ⚠️ **Hinweis:** Das Beispiel verwendet ~30.000 € als lohnsteuerpflichtiges Einkommen. Dabei handelt es sich um eine **schematische Schätzung** — das tatsächliche Wagglz-GF-Gehalt für 2025 weicht davon ab (siehe [[Wagglz GF-Gehalt 2025]]). Das Beispiel veranschaulicht den Mechanismus und stellt keine präzise Berechnung dar. Wie beide Quellen selbst anmerken: *„Genaue Berechnung durch SP STB."*
