# Strukturelles Defizit

## Definition

Ein **strukturelles Defizit** liegt vor, wenn das regelmäßige Einkommen (Fixgehalt) dauerhaft nicht ausreicht, um die laufenden Ausgaben zu decken — unabhängig von einmaligen oder unregelmäßigen Einnahmen. Das Defizit ist strukturell, weil es **nicht durch temporäre Lücken**, sondern durch dauerhaft zu hohe Fixkosten oder zu niedrige Festeinnahmen entsteht.

> Abgrenzung: Ein **Liquiditätsengpass** kann temporär sein (einmaliger Ausgaben-Spike). Ein strukturelles Defizit ist chronisch und erfordert strukturelle Maßnahmen.

## Olegs Situation (Illustration des Konzepts)

| Größe | Wert |
|---|---|
| Netto-Fixgehalt (Monat) | 3.638,82 € |
| Fixkosten gesamt | ~1.215 €/Mo |
| Strukturelles Defizit (Jahr) | **−7.970 €/Jahr** |
| Quartalskommission (netto) | ~4.611 € |
| Scheinausgleich durch Kommissionen | ~18.444 €/Jahr |
| Unternehmenslasten (Wagglz + OK Capital) | ~8.000 €/Jahr |

Das strukturelle Defizit von **−7.970 €/Jahr** bedeutet: Das Fixgehalt deckt die laufenden Ausgaben nicht. Die Differenz wird durch quartalsweise Kommissionen ausgeglichen — was das Defizit **kaschiert**, aber nicht **löst**.

---

## Das Kaschierungsproblem

Das Kernproblem eines strukturellen Defizits, das durch unregelmäßige Einnahmen überdeckt wird:

1. **Falsche Sicherheit:** Quartalsmäßige Kommissionszahlungen erzeugen periodische Liquiditätspuffer
2. **Kein Vermögensaufbau:** Die Kommissionen werden vollständig für laufende Ausgaben und Unternehmenslasten verbraucht
3. **Wachsende Vulnerabilität:** Fällt eine Kommission aus oder wird kleiner, entsteht sofort eine Liquiditätskrise
4. **Reaktives statt proaktives Finanzmanagement:** Ausgaben werden reaktiv an verfügbarer Liquidität orientiert

> *„Kommissionen erzeugen falsche Puffer — das strukturelle Defizit ist immer da, es wird nur quartalsweise überdeckt. Bank-Statement-Realität schlägt Schätzungen immer."* — [[Finanz-Rehabilitation Plan Source]]

## Ursachen des strukturellen Defizits (Olegs Kontext)

| Ursache | Beitrag |
|---|---|
| Wagglz GmbH Unternehmenslasten | Haupttreiber (~8.000 €/Jahr) |
| OK Capital UG Unternehmenslasten | Ko-Treiber (im ~8.000 € enthalten) |
| Gesellschafterdarlehen | 357 €/Mo = 4.284 €/Jahr |
| Dispo-Zinsen | Laufende Kosten durch negativen Kontostand (−1.405 € Dispo) |

---

## Lösung: Strukturelle Maßnahmen

Ein strukturelles Defizit erfordert strukturelle Maßnahmen — keine temporären Flicklösungen:

| Maßnahme | Wirkung |
|---|---|
| Unternehmenslasten eliminieren (Wagglz/OK Capital) | Entfernt den größten Kostentreiber |
| Hard Cap System (Spending-Cap) | Verhindert, dass variables Ausgabeverhalten das Defizit vergrößert |
| Kommissionen zweckgebunden allokieren | Wandelt reaktive Ausgabe in planmäßige Verteilung um |
| Gesellschafterdarlehen abbauen | Reduziert monatliche Fixlast um 357 €/Mo |
