# Hyrox Competition

## Overview

[[Hyrox Competition]] is a standardized fitness race that combines functional strength exercises with endurance running. The format consists of 8 workout stations interspersed with 1km runs, totaling 8km of running and 8 functional exercises.

## Competition Format

### Structure
- **Total Distance:** 8km running + 8 functional stations
- **Pattern:** 1km run → exercise station → 1km run → exercise station (repeat 8x)
- **Duration:** Typically 1-2 hours depending on fitness level
- **Divisions:** Open (general), Pro (elite), age groups, team formats

### Standard Workout Stations (Men's Open)

1. **1km Run → Ski Erg (1,000m)**
2. **1km Run → Sled Push (102kg)**
3. **1km Run → Sled Pull (78kg)**
4. **1km Run → Burpee Broad Jump (80m)**
5. **1km Run → Rowing (1,000m)**
6. **1km Run → Farmers Carry (2× 24kg, 200m)**
7. **1km Run → Sandbag Lunges (20kg, 100m)**
8. **1km Run → Wall Balls (6kg, 100 reps)**

## Physical Demands

### Energy System Requirements
- **Aerobic Capacity:** Primary determinant for 8km total running
- **Anaerobic Power:** Essential for high-intensity station work
- **Lactate Clearance:** Rapid recovery between anaerobic and aerobic efforts
- **Muscular Endurance:** Sustained performance over 1-2 hours

### Movement Patterns
- **Functional Strength:** All major movement patterns represented
- **Cardiovascular Endurance:** Sustained aerobic output
- **Power Endurance:** Repeated high-intensity efforts under fatigue
- **Mental Toughness:** Psychological resilience during discomfort

## Training Implications

### Hybrid Training Requirements
- **Strength Training:** Maintain power for station work
- **Endurance Training:** Build aerobic capacity for running
- **Transition Training:** Practice rapid gear switches between modalities
- **[[Brick Training]]:** Specific practice of exercise-to-running transitions

### Common Athlete Profiles
- **Strength Athletes:** Strong in stations, limited by running capacity
- **Endurance Athletes:** Excel in running, challenged by strength stations
- **CrossFit Athletes:** Balanced but may lack running specificity
- **Hybrid Specialists:** Optimally trained for format demands

## Strategic Considerations

### Pacing Strategy
- **Conservative Start:** Avoid early lactate accumulation
- **Station Efficiency:** Minimize transition time and wasted movement
- **Running Consistency:** Maintain steady pace rather than surge-and-fade
- **Energy Management:** Reserve capacity for final stations under fatigue

### Training Periodization
- **Base Building:** Aerobic foundation development
- **Specific Preparation:** Sport-specific conditioning
- **Competition Simulation:** Full-distance practice events
- **Taper:** Recovery optimization before competition

## Equipment and Logistics

### Standardized Equipment
- All weights and distances are standardized across competitions
- Equipment provided by event organizers
- Minimal personal gear required (shoes, clothing, hydration)

### Competition Environment
- **Indoor Venues:** Typically held in exhibition halls or large gyms
- **Controlled Conditions:** Consistent temperature and surface
- **Spectator Friendly:** Compact format allows viewing of entire event
