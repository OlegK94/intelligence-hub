# Password Manager Migration

## Überblick

Die Migration von [[Proton Pass]] zu [[Apple Passwords]] ist ein Trade-off zwischen:
- **Ecosystem-Integration:** Apple Passwords ist tight integriert mit iPhone/iPad/Mac
- **Portabilität:** Proton Pass funktioniert cross-platform (Android, Chrome, etc.)
- **Privacy:** Beide sind Encryption-first; Unterschiede minimal

## Entscheidungsmatrix

| Kriterium | Proton Pass | Apple Passwords |
|---|---|---|
| **Ecosystem-Integration** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Cross-Platform** | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **Privacy** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Usability auf iOS** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Portabilität** | ⭐⭐⭐⭐⭐ | ⭐⭐ |

## Status

**⏳ OFFEN** — Keine Entscheidung getroffen. Abhängig von:
1. Geplante Nutzung von Android-Geräten in Zukunft? (→ Proton Pass behalten)
2. Maximale Usability auf iOS/Mac Ziel? (→ Apple Passwords migrieren)
