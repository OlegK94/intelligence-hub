# EUIPO Trademark Registration

## Overview

**EUIPO** (European Union Intellectual Property Office) manages EU-wide trademark registration. For the [[Performance Coffee Brand]], trademark registration is a **pre-launch requirement** — must be filed before any public brand presence.

## Relevance to Performance Coffee Brand

### Why Register
- Protects brand name across all 27 EU member states with a single filing
- Blocks competitors from registering the same/similar name in the EU
- Required before any public launch (website, social media, events)
- Enables legal enforcement if others use the mark

### Classes Required

| Nice Class | Category | Relevance |
|---|---|---|
| **Class 30** | Coffee, tea, functional food, instant beverages | Primary product class |
| **Class 5** | Dietary supplements (NEM), nutraceuticals | If sold as NEM |

Both classes are needed given the dual positioning (coffee + functional supplement).

## Process & Cost

- **Cost:** ~1.000 € for 2 Nice Classes
- **Route:** Direct via euipo.europa.eu OR via trademark attorney
- **Platform:** EUIPO eSearch Plus — for prior art search
- **Timing:** File BEFORE first public appearance

## Pre-Registration Steps

1. **Availability search:** Check chosen name (AEVUM / KOLVR / ORVYN) on EUIPO eSearch Plus
2. **Domain check:** namecheap.com or instant.domains for .com availability
3. **Decision:** Finalize brand name
4. **File:** Submit application via euipo.europa.eu

## Timeline

- EUIPO typically takes **4–6 months** for standard examination
- Opposition period: 3 months after publication
- Registration valid for **10 years** (renewable)
