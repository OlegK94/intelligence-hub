## Aufgabe 2: Wagglz Strategieentscheidung treffen

### Deadline: 2026-07-01

### Situation
- **Umsatz:** Seit Januar 2026 keine Einnahmen
- **Status:** Entscheidung ausstehend: Weiterführen (mit neuem Konzept) vs. Auflösen

### Option A: Weiterführen (mit neuen Konzept)
- Erfordert ein profitables oder vielversprechendes Geschäftsmodell
- [[Wufflz]] (B2B Tier-Management) als potenzielle Wiederaktivierung?
- Weitere Entwicklungskosten
- Jahresabschluss-Kosten: €5.000 (aktuell), €2.500 (nächstes Jahr)

### Option B: Auflösen
- **Vorteil:** §17 EStG Verlustveräußerung
  - Wagglz GmbH Betriebsvermögen als Sacheinlage → Veräußerung löst Einkommensteuer-Verlust aus
  - Verlust (~€150k Verlustvortrag) wird in der persönlichen ESt nutzbar
  - Potenzielle Steuerersparnis: ~€50k–60k (abhängig von Grenzsteuersatz)
- **Auflösungskosten:** Notargebühren, Handelsregister-Eintrag, finale Abschlüsse
- **Zeitaufwand:** Insolvenz-Antrag vs. ordentliche Auflösung

### Laufende Kosten (falls Weiterführung)
| Position | Betrag/Monat |
|---|---|
| Oleg-Darlehen-Rückzahlung | €120–255 |
| Jahresabschluss (auf Jahresbasis) | ~€417 |
| **Summe** | **~€537–672/Monat** |

**Problem:** Bei 0 € Umsatz ist jeder Monat ein Verlust.

### Entscheidungspfad


## Aufgabe 3: Fixkosten-Optimierung prüfen

### Deadline: 2026-07-01

### Aktuelle Situation
- **Netto-Gehalt (durchschnittlich):** ~€8.250/Monat
- **Fixkosten (bekannt):** ~€1.215/Monat (61% des Nettogehalts) ⚠️
- **Ziel:** < 50% der Netto-Einnahmen

### Zu überprüfende Positionen

#### 1. Apple-Abos
| Abo | Kosten | Überprüfung |
|---|---|---|
| Apple One (iCloud, Music, etc.) | ~45 €/Monat | Welche Services werden tatsächlich genutzt? |
| | | → Reduzierte iCloud? (nur 50 GB = 0,99 €) |
| | | → Musik statt Apple Music? (Spotify Family?) |

**Ersparnis-Potenzial:** €20–30/Monat (wenn nicht genutzten Abos gekündigt)

#### 2. YouTube Premium
| Position | Kosten |
|---|---|
| YouTube Premium | 23,99 €/Monat |

**Frage:** Ist Premium notwendig oder kann reguläres YouTube ausreichen?

**Ersparnis-Potenzial:** €23,99/Monat (falls nicht essentiell)

#### 3. Uelzener Tierversicherung
| Position | Kosten |
|---|---|
| Tierversicherung (Uelzener) | 132,48 €/Monat |

**Überprüfung:**
- Ist eine Alternative mit besserer Quote vorhanden? (Vergleichsportale prüfen)
- Erhöhte Selbstbeteiligung akzeptabel?
- Kann Deckung reduziert werden?

**Ersparnis-Potenzial:** €30–50/monat (abhängig von Anbieter-Wechsel)

### Aktuelle Fixkosten-Übersicht

| Position | Betrag/Monat |
|---|---|
| VW-Bank Kredit | €681,57 |
| Kieferorthopädie (Dr. Wiemann) | €176,53 |
| Gesellschafterdarlehen → Unternehmen | ~€357,00 |
| Apple-Abos | ~€45,00 |
| YouTube Premium | €23,99 |
| Uelzener Tierversicherung | €132,48 |
| **Summe (bekannt)** | **~€1.416,57** |

**Struktur-Problem:** Selbst nach Optimierung bleiben ~€860/Monat (VW-Bank + Kieferorthopädie + Gesellschafterdarlehen) als unvermeidbare Kosten → immer noch 10% über der 50%-Zielschwelle.

### Ersparnis-Szenarien

| Szenario | Optimierungen | Neue Fixkosten | % Netto |
|---|---|---|---|
| **Status Quo** | keine | €1.416,57 | 17,2% |
| **Konservativ** | Apple (-€20), YouTube (-€24) | €1.372,57 | 16,6% |
| **Aggressiv** | Apple (-€30), YouTube (-€24), Uelzener (-€40) | €1.322,57 | 16,0% |
| **Ziel-Szenario** | Aggressive + Haushalt-Überprüfung | <€1.200 | <14,5% |

### Überblick der erforderlichen Maßnahmen

- [ ] Apple-Abo-Details überprüfen; ggf. auf iCloud-only reduzieren
- [ ] YouTube Premium Notwendigkeit prüfen; ggf. kündigen
- [ ] Uelzener-Alternative recherchieren (Allianz, AXA, andere Versicherer)
- [ ] Alle Optimierungen mit [[SP STB]] oder Finanzplanung koordinieren
