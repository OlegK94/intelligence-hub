# Clevr Blends

## Überblick

**Clevr Blends** ist eine New Yorker Wellness-Latte-Brand, die plant-based Instant-Lattes mit Adaptogenen kombiniert. Das Besondere: Hafermilch ist bereits im Produkt integriert, was es kalorienreicher macht als reine Functional-Coffee-Produkte.

## Eckdaten

| Parameter | Wert |
|---|---|
| **Sitz** | New York, NY |
| **Preis/Portion** | ~$1,00 (Abo: $23,80/28 Srv.) |
| **Preis regulär** | $28/Beutel (28 Portionen) |
| **EU-Präsenz** | **Nein** |

## Produktportfolio

| Produkt | Inhalt |
|---|---|
| **Coffee SuperLatte** | Arabica Coffee + Ashwagandha + Lion's Mane + Reishi + Probiotika + Oat Milk |
| **Matcha SuperLatte** | Matcha + Adaptogene + Oat Milk |
| **Golden SuperLatte** | Kurkuma + Adaptogene + Oat Milk |

## Stack-Tiefe

| Dimension | Bewertung |
|---|---|
| Kaffee | Ja (Arabica) |
| Pilze | Lion's Mane, Reishi |
| Adaptogene | Ashwagandha |
| Longevity-Ingredients | **Nein** |
| Dosierung | Sehr flach; niedrige mg-Gehalte |
| **Oat Milk integriert** | **Ja** — kalorienhaltig |

## Schwächen

1. Kalorienhaltig durch integrierte Hafermilch — nicht für „clean" Keto/Low-Carb-Nutzer
2. Sehr flacher Wirkstoffstack; niedrige Dosierungen
3. Kein wissenschaftlicher Longevity-Claim
4. Keine EU-Präsenz
