# Wufflz Pitch Investoren

## Überblick

**Wufflz Pitch Investoren** bezeichnet drei Versionen (v1–v3) des Investoren-Pitchdecks für [[Wagglz GmbH]] / [[Wufflz]], die in Figma erstellt wurden. Die drei Versionen deuten auf einen iterativen Entwicklungsprozess hin — möglicherweise für verschiedene Investitionsphasen, unterschiedliche Zielgruppen oder als Reaktion auf Feedback aus frühen Investorengesprächen.

> Die vollständigen File Keys sind im README nur als Präfixe angegeben (`5Fp2…`, `Fnhm…`, `1gag…`) — vollständige Keys im [[Figma Index Wagglz]].

## Strategische Einordnung

Drei Versionen eines Investoren-Pitchdecks deuten auf:
- **Iterativer Pitch-Prozess** — Deck wurde basierend auf Feedback oder Gesprächsergebnissen weiterentwickelt
- **Unterschiedliche Zielgruppen** — v1/v2/v3 könnten für verschiedene Investorentypen (Angels, VCs, Strategische Investoren) angepasst sein
- **Aktive Fundraising-Phase** — mehrere Investorengespräche fanden statt oder waren geplant
- **Professionelle Aufbereitung** — Design in Figma statt nur PowerPoint/Keynote zeigt Design-Qualitätsanspruch

Das Vorhandensein von sowohl Banken-Pitch ([[Wufflz Pitch Deck Banken]]) als auch drei Investoren-Versionen zeigt, dass Wagglz GmbH eine **Multi-Channel-Finanzierungsstrategie** verfolgte.

> ⚠️ **Kontext-Hinweis:** Laut [[ARAG Wagglz Versicherung]] und anderen Quellen steht eine mögliche Schließung der Wagglz GmbH im Raum. Der Ausgang der Finanzierungsgespräche, für die diese Decks erstellt wurden, ist unklar.
