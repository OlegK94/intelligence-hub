# Hawlik BioImport GmbH

## Überblick

**Hawlik BioImport GmbH** (Langweid am Lech, Bayern) ist nach eigenen Angaben Europas führender Hersteller für Nahrungsergänzungsmittel aus Vitalpilzen. Für die [[Performance Coffee Brand]] ist Hawlik der **Schlüsselpartner für Lion's Mane + Cordyceps Dual-Extrakte** — sowohl als Rohstofflieferant als auch als potenzieller CMO für pilzbasierte Sachet-Komponenten.

## Eckdaten

| Parameter | Details |
|---|---|
| **Standort** | Langweid am Lech, Bayern |
| **Website** | bioimport-eu.de |
| **Erfahrung** | 20 Jahre Vitalpilz-Analytik |
| **Zertifizierungen** | GMP (Deutschland + Österreich) |
| **Qualitätslabor** | AGROLAB (staatlich akkreditiert, unabhängig) |
| **CMO-Rolle** | Rohstoff + ggf. Contract Manufacturing |

## Produktportfolio

- **Eigene Produktlinie:** Hawlik Vitalpilze (Beweis der eigenen Qualitätsstandards)
- **Pilzextrakte:** Lion's Mane, Cordyceps, Reishi, Shiitake u.a.
- **Services:** Private Label, Formulierung, Kapselabfüllung, Verpackung

## Qualitätsstandards

| Kriterium | Hawlik-Standard |
|-----------|----------------|
| Extraktion | Dual-Extrakt (Heißwasser + Ethanol) |
| Analytik | AGROLAB (staatlich akkreditiert) |
| COA | Für jeden Batch |
| GMP | DE + AT |
| Schwermetalle | Im COA enthalten |

## Relevanz für Performance Coffee Brand

### Pilzextrakt-Komponenten

- **Lion's Mane** (Hericium erinaceus): Dual-Extrakt für Beta-Glucane + Hericenone
- **Cordyceps** (C. militaris / C. sinensis): Dual-Extrakt für Beta-Glucane + Cordycepin
- Beide Extrakte im Format für Sachet-Mischungen verfügbar

### CMO-Potenzial

Hawlik kann nicht nur Rohstoffe liefern, sondern auch als CMO fungieren — eine pilzbasierte Sachet-Variante bei Hawlik produzieren lassen ist prinzipiell möglich.
