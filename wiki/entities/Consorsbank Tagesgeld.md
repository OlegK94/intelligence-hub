# Consorsbank Tagesgeld

## Übersicht

[[Consorsbank]] Tagesgeldkonto ist die designierte Sparstätte für den [[Notfall-Puffer (Oleg)]]. Es bietet zum aktuellen Datum (16.06.2026) eine Verzinsung von 1,00% p.a. und wird zur Finanzierung der Phase-1-Notfall-Rücklagen genutzt.

## Kontodetails

| Parameter | Wert |
|---|---|
| **Anbieter** | Consorsbank AG |
| **Kontoart** | Tagesgeld |
| **Zinssatz** | 1,00 % p.a. (ab 16.06.2026) |
| **Automatische Verzinsung** | ✅ Täglich / monatlich |
| **Zieleinzahlung** | €10.900 (3× Nettoeinkommen) |
| **Erste Finanzierungsquelle** | Q2-Bonus Juli 2026 (~€8.250 netto) |
| **Aufbau-Timeline** | Phase 1: 1–3 Monate |

## Nutzen

- **Notfall-Puffer:** Deckt 3 Monate unerwartete Ausgaben ab
- **Liquidität:** Tagesgeld-Konten haben typisch 1–2 Arbeitstage Abhebungsfrist
- **Zinsertrag:** 1% p.a. auf €10.900 = ~€109/Jahr bei vollständiger Finanzierung
