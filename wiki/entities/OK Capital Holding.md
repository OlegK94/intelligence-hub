# OK Capital Holding

## Überblick

**OK Capital Holding** ist die IP-Holding-Gesellschaft für das [[Performance Coffee Brand]]-Vorhaben. Sie hält die Marke sowie die Rezeptur-/Formulierungs-IP in einer steueroptimierten Struktur, getrennt von der operativen [[Wagglz GmbH]].

## Rolle in der Performance Coffee Brand-Struktur

| Gesellschaft | Rolle |
|---|---|
| [[Wagglz GmbH]] | Betriebsgesellschaft (OpCo) |
| **OK Capital Holding** | IP-Holdinggesellschaft (Marke + Rezeptur) |

## Begründung für die IP/OpCo-Trennung

Die Trennung von IP und Betrieb ist eine gängige Steueroptimierungsstrategie:
1. **IP auf Holding-Ebene** → IP-Einkünfte (Lizenzgebühren von der OpCo) können zum niedrigeren Holding-Steuersatz versteuert werden
2. **OpCo lizenziert IP von der Holding** → Betriebskosten (Lizenzgebühren) sind auf OpCo-Ebene abzugsfähig
3. **Vermögensschutz** → IP-Vermögenswerte sind vor operativer Haftung geschützt
4. **Umzugskompatibel** → konsistent mit der [[Cyprus Relocation]]-Strategie; zieht Oleg nach Zypern um, können Einkünfte aus IP auf Holding-Ebene vom günstigen zypriotischen IP-Regime profitieren
