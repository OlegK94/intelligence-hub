# Finanz-Rehabilitation — 3-Phasen-Plan

## Overview

A structured 3-phase financial rehabilitation plan for [[Oleg Personal Context|Oleg]], aimed at transitioning from a structural annual deficit to stable liquidity. The core principle: commission income will be allocated according to a fixed schema — no longer consumed reactively.

> For the full source document, see [[Finanz Rehabilitation Plan Source Detail]].

## Kernproblem (Root Problem)

| Parameter | Wert |
|---|---|
| **Strukturelles Jahresdefizit** | −7.970 €/Jahr (Privat) |
| **Quartalskommissionen** | ~4.611 €/Quartal = ~18.444 €/Jahr |
| **Unternehmenslasten** | ~8.000 €/Jahr (Wagglz + OK Capital) |
| **Ergebnis** | Kein Aufbau von Reserven oder Vermögen möglich |

### Structural Mechanism

The structural deficit is permanently **masked** — not solved — by quarterly commissions:

```
Fixgehalt → reicht NICHT für laufende Ausgaben
    ↓
Kommissionen → gleichen Defizit aus
    ↓
Unternehmenslasten (~8.000 €/Jahr) → fressen Kommissionen weiter auf
    ↓
Ergebnis: kein Vermögensaufbau möglich
```

> **Lernprinzip:** Kommissionen erzeugen falsche Puffer — das strukturelle Defizit ist immer da, es wird nur quartalsweise überdeckt. Bank-Statement-Realität schlägt Schätzungen immer.

### Wurzelproblem

[[Wagglz GmbH]] + [[OK Capital UG]] are the structural cost block. Resolution of these entities is a **prerequisite** for sustainable financial stability.

## Phase 1 — Sofort-Überleben (Aktuell)

**Ziel:** Dispo stabilisieren, keine neuen Schulden, Überblick schaffen

- [ ] Bankkontoauszüge (CSV) exportieren — vollständige Ist-Analyse
- [ ] Monatliches Spending-Cap definieren (Ziel: unter Netto-Gehalt bleiben)
- [ ] Jahresabschlüsse-Kosten (5.000 €) einplanen
- [ ] Insolvenzstatus beider Gesellschaften klären (→ Anwalt/Steuerberater)

**Status:** Nächste Quartalskommission (Juli) wird fast vollständig durch Accountant-Fees absorbiert.

## Phase 2 — Juli-Reset (Zieldatum: August 2026)

**Ziel:** Saubere Allokation der Juli-Kommission

| Verwendung | Betrag | Priorität |
|---|---|---|
| Accountant-Fees (anteilig) | ~2.500 € | 1 — rechtliche Pflicht |
| Dispo ausgleichen (−1.405 €) | 1.405 € | 2 — Zinskosten |
| Notfall-Puffer | 700 € | 3 — Mindestpuffer |
| Reserve für Defizit-Abfederung | ~0 € | — |

> **Problem:** Die Provision reicht kaum. Zeigt Dringlichkeit der Unternehmens-Bereinigung.

## Phase 3 — Hard Cap System (ab Q4 2026)

**Ziel:** Strukturelles Defizit eliminieren durch harte monatliche Ausgabenobergrenze

| Parameter | Wert |
|---|---|
| **Netto-Gehalt** | 3.638,82 €/Mo |
| **Fixkosten gesamt** | ~1.215 €/Mo |
| **Verbleibend (variabel)** | ~2.424 €/Mo |
| **Puffer/Sparen-Ziel** | ≥300 €/Mo |

**Voraussetzung:** Gesellschafterdarlehen (357 €/Mo) müssen abgebaut oder umstrukturiert werden.
