# Finanz-Rehabilitation — 3-Phasen-Plan

## Überblick

Ein strukturierter 3-Phasen-Finanz-Rehabilitationsplan für [[Oleg Personal Context|Oleg]], der den Übergang von einem strukturellen Jahresdefizit zu stabiler Liquidität zum Ziel hat. Das Kernprinzip: Kommissionseinnahmen werden nach einem festen Schema zugeteilt — nicht länger reaktiv verbraucht.

> Das vollständige Quelldokument findet sich unter [[Finanz Rehabilitation Plan Source Detail]].

## Kernproblem

| Parameter | Wert |
|---|---|
| **Strukturelles Jahresdefizit** | −7.970 €/Jahr (Privat) |
| **Quartalskommissionen** | ~4.611 €/Quartal = ~18.444 €/Jahr |
| **Unternehmenslasten** | ~8.000 €/Jahr (Wagglz + OK Capital) |
| **Ergebnis** | Kein Aufbau von Reserven oder Vermögen möglich |

### Struktureller Mechanismus

Das strukturelle Defizit wird durch Quartalskommissionen dauerhaft **kaschiert** — nicht gelöst:

```
Fixgehalt → reicht NICHT für laufende Ausgaben
    ↓
Kommissionen → gleichen Defizit aus
    ↓
Unternehmenslasten (~8.000 €/Jahr) → fressen Kommissionen weiter auf
    ↓
Ergebnis: kein Vermögensaufbau möglich
```

> **Lernprinzip:** Kommissionen erzeugen falsche Puffer — das strukturelle Defizit ist immer da, es wird nur quartalsweise überdeckt. Die Realität der Kontoauszüge schlägt Schätzungen immer.

### Wurzelproblem

[[Wagglz GmbH]] + [[OK Capital UG]] sind der strukturelle Kostenblock. Die Auflösung dieser Gesellschaften ist eine **Voraussetzung** für nachhaltige finanzielle Stabilität.

## Phase 1 — Sofort-Überleben (Aktuell)

**Ziel:** Dispo stabilisieren, keine neuen Schulden, Überblick schaffen

- [ ] Bankkontoauszüge (CSV) exportieren — vollständige Ist-Analyse
- [ ] Monatliches Spending-Cap definieren (Ziel: unter Netto-Gehalt bleiben)
- [ ] Jahresabschlusskosten (5.000 €) einplanen
- [ ] Insolvenzstatus beider Gesellschaften klären (→ Anwalt/Steuerberater)

**Status:** Die nächste Quartalskommission (Juli) wird fast vollständig durch Buchhalterhonorare absorbiert.

## Phase 2 — Juli-Reset (Zieldatum: August 2026)

**Ziel:** Saubere Allokation der Juli-Kommission

| Verwendung | Betrag | Priorität |
|---|---|---|
| Buchhalterhonorare (anteilig) | ~2.500 € | 1 — rechtliche Pflicht |
| Dispo ausgleichen (−1.405 €) | 1.405 € | 2 — Zinskosten |
| Notfall-Puffer | 700 € | 3 — Mindestpuffer |
| Reserve für Defizit-Abfederung | ~0 € | — |

> **Problem:** Die Provision reicht kaum. Zeigt die Dringlichkeit der Unternehmensbereinigung.

## Phase 3 — Hard Cap System (ab Q4 2026)

**Ziel:** Strukturelles Defizit durch harte monatliche Ausgabenobergrenze eliminieren

| Parameter | Wert |
|---|---|
| **Netto-Gehalt** | 3.638,82 €/Mo |
| **Fixkosten gesamt** | ~1.215 €/Mo |
| **Verbleibend (variabel)** | ~2.424 €/Mo |
| **Puffer/Sparziel** | ≥300 €/Mo |

**Voraussetzung:** Gesellschafterdarlehen (357 €/Mo) müssen abgebaut oder umstrukturiert werden.

## Ausstehende Schlüsselentscheidungen

1. **[[Wagglz GmbH]] + [[OK Capital UG]]:** Liquidation, Ruhendstellung oder Weiterführung?
2. **VW-Bank-Kredit** (681,57 €/Mo bis 11/2028): Sondertilgung möglich/sinnvoll?
3. **Steuerliche Verlustvorträge:** Können gegen Privateinkommen gerechnet werden?
